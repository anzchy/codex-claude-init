Startup Flow

  1. User double-clicks Reader3.app
  2. Tauri app spawns Python sidecar with env vars (READER3_BOOKS_DIR, READER3_AI_SETTINGS)
  3. Sidecar Manager polls GET http://127.0.0.1:8123/ every 500ms
  4. WebView shows loading indicator during health check
  5. On 200 response, WebView navigates to http://127.0.0.1:8123/
  6. FastAPI serves library.html via Jinja2, existing JS/CSS loads in WebView

Shutdown Flow

  1. User closes app (Cmd+Q or window close)
  2. Tauri sends SIGTERM to sidecar process
  3. Wait 2 seconds for graceful shutdown
  4. If still running, send SIGKILL
  5. App window closes, all processes terminated
